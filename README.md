# Overdracht-tool — Sportverenigingen v2.1

Interne overdrachtstool met login, acties-systeem, CSV-upload en gemeente-dashboard.

## Nieuw in v2.1
- **CSV-upload** bij Verenigingen: sjabloon downloaden, invullen, bulk-uploaden
- **Acties per club**: snelle notities + volwaardige acties met status/deadline/verantwoordelijke
- **Gemeente-dashboard**: cockpit met tegels, statistieken, recente activiteit en tekstvelden
- **Loginscherm** met 3 accounts (Admin, Justin, Amee)

## Data
Alle data in `data/` map — wordt via GitHub API bijgewerkt vanuit de tool.

## Setup
1. Upload naar GitHub, activeer GitHub Pages
2. Log in als Admin, ga naar Instellingen, koppel je GitHub-token
3. Klaar!
